# Parking Spot Tracker

아파트 주차장 도면에 내가 주차한 위치를 날짜별로 저장하고 다시 확인하는 개인용 웹앱입니다.

## 기능

- 주차장 도면 이미지 업로드 (로컬 브라우저 저장)
- 도면 클릭으로 주차 위치 핀 표시
- 날짜별 위치/층/메모 저장
- 최근 기록 목록 확인
- 모든 데이터는 브라우저 `localStorage`에 저장

## 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 도면 이미지 업로드
3. 도면 클릭으로 주차 위치 지정
4. 필요 시 층/메모 입력 후 저장

## 참고

- 같은 브라우저, 같은 PC 기준으로 데이터가 유지됩니다.
- 브라우저 데이터를 삭제하면 기록도 함께 삭제됩니다.
